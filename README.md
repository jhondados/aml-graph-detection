# 💰 Anti-Lavagem de Dinheiro (AML)

> Detecção de lavagem de dinheiro com análise de grafos. Identifica estruturas suspeitas, smurfing e contas laranja em redes de transações.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/aml-graph-detection/

[![Vertical](https://img.shields.io/badge/Vertical-Finanças-blue)]() [![IA](https://img.shields.io/badge/IA-Graph%20ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**94.1%** Detecção · **247** Anéis · **1.840** SAR gerados · **2.1%** Falso Pos.

## 🧠 Tecnologias de IA
Graph ML · GNN · Anomaly Detection

## ✨ O que faz
- Análise de grafo de transações
- Detecção de smurfing/estruturação
- Geração automática de SAR
- Score de risco por entidade

## 🗄️ Fonte de dados real
[COAF + Banco Central](https://www.gov.br/coaf)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com COAF + Banco Central
2. **Processamento com IA** — Graph ML · GNN · Anomaly Detection
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
